## 简介

这是一个基于Python的局域网文件传输工具。

～～已在本机完成过测试，但由于没有多余的电脑，所以没有试过在局域网内的传输，不过理论上应该是可行的～～

为了追求打包的轻量化，采用了tk架构而非更顺眼且更好调的pyqt。网络传输方面使用socket。～～这两方面我都比较陌生，因此不保证通用的可用性～～

## 优化方向

  1. gui优化，生成更好看的gui
